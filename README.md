# docker_example

### Install libraries

```
pip install -r requirements.txt
```


### Make a docker image

```
./sh/deploy.sh
```

### Run a docker image

```
./sh/run.sh
```

### Stop a docker container

```
./sh/stop.sh
```

### Start a docker container

```
./sh/start.sh
```

### Test with a client

```
python client.py
```
